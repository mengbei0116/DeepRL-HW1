# GridWorld 強化學習互動環境 (DeepRL-HW1)

這是一個以 Python Flask 結合前端網頁技術 (HTML, CSS, JavaScript) 所開發的 GridWorld 強化學習互動環境。本專案的主要目的是視覺化地展示「環境的設定」、「初始狀態」、以及透過「價值迭代 (Value Iteration)」演算法求解的動態擴展過程與最終最佳路徑結果。
*  DEMO網站:https://deep-rl-hw-1.vercel.app/

## 1. 網格地圖開發

*   **視覺化科技風格**：網頁整體採用暗色系主視覺，並搭配青色與霓虹發光效果 (Cyan Glow) 的科技風格，創造具深度與沈浸感的操作體驗。
*   **動態網格生成與固定版面**：允許使用者透過滑桿即時切換 $N \times N$ 的網格大小 ($N \in [5, 9]$)。同時，網頁具有預先保留空間的外部容器，確保內部格子重繪時網頁大小不會反覆跳動與浮動。在保有固定的 `.cell` 大小下，也能展現動態的版面彈性設計。
*   **互動式操作體驗**：
    *   **依序設定元素**：使用者可非常直覺地在網格中利用滑鼠點擊，依序部署「起始點（綠色）」、「終點（紅色）」以及剛好 $N-2$ 個「障礙物（灰色）」方塊。
    *   **動態狀態提示與懸停預覽 (Hover Preview)**：前端除了透過文字即時顯示目前應放入何種方塊外，當游標移至空白格子時，也會根據目前擺放階段，智慧渲染該步驟對應的半透明發光顏色做為防錯預覽，輔助使用者操作。

## 2. 策略與價值顯示

*   **無縫的矩陣檢視切換**：當環境障礙物設定完畢（共 $N-2$ 個）的 $0.5$ 秒後，原先的手動設定網頁將會自動隱藏，取而代之地在最前方精美並排渲染出全新兩組結果矩陣：「**Value Matrix (狀態價值矩陣)**」與「**Policy Matrix (策略矩陣)**」。
*   **直觀的環境標示**：矩陣會自動繼承起點（深綠框）、終點（紅底，目標 `G`）與障礙物（深灰，標記 `X`）的位置。其餘所有的普通狀態在初始時，會建立精確到小數點後兩位（$0.00$）的價值數值預覽，以及各個獨立格子的隨機方向箭頭（`↑`, `↓`, `←`, `→`）。
*   **即時重置能力**：使用者僅需簡單再次拉動滑桿改變大小，$N$ 與網格即可毫無延遲地初始化重置，自動切換回最初的網格設定介面。

## 3. 使用價值迭代算法推導最佳政策

*   **深度防呆與連通性檢測**：迭代後若檢測到障礙物完全封死路線而無法到達終點時，會跳出警告提示。
*   **Value Iteration 擴展動畫**：按下開始迭代按鈕後，會呼叫 Flask API 即刻開始運算。在設定抵達終點的 Reward 為 $+1.0$、折扣因子 $\gamma = 0.9$ 的動態環境下反覆運算直到 $\Delta < 10^{-4}$ 收斂。演算法將每個迭代 (Iteration) 的矩陣狀態傳回前端，並採用 $250$ 毫秒/次的定時更新頻率，渲染出一幕幕價值如何從終點擴散渲染開來的「數值傳遞視覺化」動畫效果。
*   **最佳路線的螢光捕捉**：一旦全區域迭代完成且成功尋得最佳解，網頁便會按照最終 Policy Matrix 的方向，從起點一步一步繪製出一條通往終點的最佳路徑，並直接將醒目的**螢光黃色外框與發光特效**重疊在對應格子上。由於設計邏輯跳過了起終點的上色，該路徑完美凸顯了最佳動向而不破壞原有的顏色指標。
